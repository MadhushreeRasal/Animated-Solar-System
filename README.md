# Animated Solar System

An animated solar system simulation built using **HTML and CSS**. This project visually demonstrates the motion of the Sun, the eight planets, and Earth's Moon through CSS animations, positioning, and transformations. It was created to practice working with layouts, animations, and nested elements while building an interactive space-themed project.

## Live Demo at

[https://madhushreerasal.github.io/Animated-Solar-System/](https://madhushreerasal.github.io/Animated-Solar-System/)

## Features

* Central Sun with continuous rotation animation
* All eight planets displayed with individual orbital paths
* Earth with a revolving Moon
* Planetary rotation and orbital revolution effects
* Smooth CSS animations using `@keyframes`
* Space-themed background for an immersive appearance
* Built using CSS positioning, transforms, and animations

## Technologies Used

* HTML5
* CSS3

  * Flexbox
  * Absolute Positioning
  * CSS Animations
  * CSS Transforms
  * Keyframes

## What I Practiced

Through this project, I gained hands-on experience with:

* Creating complex layouts using nested elements
* Working with relative and absolute positioning
* Building circular orbital paths
* Applying multiple CSS animations simultaneously
* Understanding parent-child animation behavior
* Organizing HTML and CSS for larger visual projects

## Purpose of the Project

This project was created as a frontend practice exercise to explore CSS animations and positioning in a creative way. Rather than building a standard webpage, the goal was to simulate planetary motion and gain a deeper understanding of how animated elements interact within a layout.
